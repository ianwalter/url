# @ianwalter/url
> The URL constructor / class without tears

[![npm page][npmImage]][npmUrl]
[![CI][ciImage]][ciUrl]

## Installation

```console
yarn add @ianwalter/url
```

## Usage

```js
const createUrl = require('@ianwalter/url')

const url = createUrl('https://ianwalter.dev')

url.hostname //=> ianwalter.dev
```
